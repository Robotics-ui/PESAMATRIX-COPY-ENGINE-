# PesaMatrix Design Preferences

## Visual Reference
- The UI must closely resemble the TradePro SaaS screenshot provided by the user (attached_assets/file_000000009e78720cb72f7bde42e2306c_1781158221869.png).
- Dark background theme with sidebar navigation.
- Cards for stats (Total Signals, Win Rate, Active Plan, Total Profit).
- Live market overview panel + performance chart.
- Recent signals list with BUY/SELL badges.
- Subscription status panel.

## Color Rule
- **All purple colors must be replaced with green.**
- Primary accent: green (not purple/violet).
- Active sidebar item: green background (not purple).
- Buttons, badges, highlights: green.
- Chart line: green.
- Crown/VIP badge: green.

**Why:** User explicitly stated "All purple colors be green" when referencing the design screenshot.

**How to apply:** Wherever the reference design uses purple (#7C3AED, #6D28D9, violet, indigo, etc.), substitute with green (#16A34A, #22C55E, or similar). Keep dark navy/slate backgrounds as-is.
